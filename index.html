Projects

Image Classification with CNN

Proyek ini melakukan klasifikasi gambar bunga (multi-class) menggunakan Convolutional Neural Network (Keras/TensorFlow). Proses pengembangan model mencakup data augmentation untuk meningkatkan variasi data latih, visualisasi learning curve untuk memantau proses pelatihan, serta analisis performa model menggunakan confusion matrix dan beragam metrik evaluasi seperti akurasi, presisi, dan recall. Hasil akhir menunjukkan model mampu mengenali berbagai jenis bunga secara efektif, serta memberikan insight terhadap kekuatan dan kelemahan model pada masing-masing kelas.

QS World University Rankings Data Analysis

Analisis data QS World University Rankings 2026 menggunakan Python. Eksplorasi meliputi pembersihan data, visualisasi heatmap missing value, analisis perubahan peringkat, serta identifikasi universitas dengan lonjakan ranking tertinggi. Ditemukan bahwa overall score sangat memengaruhi peringkat, dengan tren kenaikan signifikan pada universitas di Asia.
